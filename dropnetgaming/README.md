# DNG CS2 Match Hub

Готовый статический проект для GitHub Pages.

## Что есть

- Многостраничный сайт через hash-роутинг.
- Старый тёмный интерфейс DNG: topbar, sidebar, rightbar, карточки матчей.
- Убраны разделы: О проекте, Дизайн-документ, Прототип, Партнёрам, Roadmap.
- Авторизация и регистрация через localStorage.
- Админ-панель: новости, обращения, пользователи.
- Новости, задания, профиль, лидерборд, Premium, турниры, команды.
- Мини-игра в отдельном окне браузера при нажатии Найти матч.
- WASD и стрелки, музыка, настройки игры, победа/поражение меняют ELO.
